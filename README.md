## Manual Install

1. Clone the repository:
```bash
git clone https://github.com/ChrisBurkett/claudestrike.git ~/claudestrike
cd ~/claudestrike
```

2. Run the installer:
```bash
chmod +x install.sh
./install.sh
```

## Requirements

- Kali Linux (or Debian-based system)
- Python 3.8+
- Anthropic API key ([Get one here](https://console.anthropic.com))

## Usage

Launch ClaudeStrike from your applications menu or run:
